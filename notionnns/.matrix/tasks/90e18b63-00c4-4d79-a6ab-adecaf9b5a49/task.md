# List Notion spaces/pages accessible to the integration

Using the Notion API connection, retrieve and list all top-level spaces, pages, and databases that the integration currently has access to. For each item, include its title, type (page/database), and ID/URL. Present the results as a clear list or table. If the Notion API distinguishes 'workspaces' differently from pages/databases, clarify that distinction in the output. No further scope beyond listing — do not create or modify any content.
