# Report current weather in Berlin, Germany

Fetch the current weather conditions for Berlin, Germany. Report: temperature (current, feels-like), conditions (e.g. clear/cloudy/rain), humidity, wind speed/direction, and any relevant alerts. Save the result as a concise summary (a few lines, human-readable) — no need for historical data or forecasts beyond current conditions. Output should be suitable to paste directly into a status note or chat reply.
